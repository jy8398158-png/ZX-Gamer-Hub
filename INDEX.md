## 🎮 HOW IT WORKS

```
1. User opens skill-assessment.html
2. Fills out the form
3. Clicks "Submit Assessment"
4. Form validates (client-side)
5. Sends data to backend
6. Backend calculates score
7. Backend determines rank
8. Backend formats Discord embed
9. Sends to Discord webhook
10. User sees success message ✅
11. You get Discord notification 📬
```
